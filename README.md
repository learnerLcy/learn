# learn(博客系统)

------
> * 基于**SpringBoot+Mybatis**通用**mapper+layui**开发,持续开发，持续学习，一步步提高。

## 目前拥有：
- [x] 前端**layui、ztree等**基本样式
- [x] 支持**Mybatis通用mapper**的增删改查，**mybatis-generator逆向工程代码自动生成**
- [x] **druid**数据库监控、**swagger2**构建api文档
- [x] **MongoDb**数据库

## 现阶段功能模块：
- [ ] 系统首页
- [x] 菜单管理
- [x] 文章管理
- [ ] 用户中心
- [x] 系统设置
- [x] 安全监控
> 目前正在持续开发过程中，其他模块以及技术，陆续增加。
